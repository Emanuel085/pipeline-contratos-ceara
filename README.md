# Pipeline de Contratos Públicos - Ceará

Este projeto implementa um pipeline de dados utilizando Apache Airflow.

## 🔧 Tecnologias utilizadas

* Python
* Apache Airflow
* PostgreSQL
* Docker
* API Ceará Transparente
* Google Gemini (IA)

## 🚀 Funcionalidades

* Extração de dados da API pública
* Armazenamento em banco PostgreSQL
* Classificação automática com IA
* Geração de relatório HTML

## 📄 Saída

* Relatório HTML com os 30 maiores contratos classificados

## ▶️ Execução

Pipeline orquestrado via DAG no Airflow.
